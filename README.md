# Data Structure Projects (Java)

## Projects

Gravitational physics simulator modeling planetary motion using pairwise force calculations and time-step updates.

### Deque Implementations

### Gitlet Version Control
A miniature Git system supporting commits, branching, checkout, and logging with persistent file snapshots.
